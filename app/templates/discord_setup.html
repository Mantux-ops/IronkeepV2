Step 1 — Add the bot to your server

Click below to invite the Ironkeep bot. You need the Manage Server permission in the Discord server to install bots.

The bot requests these permissions: Send Messages, Embed Links, Read Message History, View Channel. No administrator access is required.

Add Ironkeep Bot to Discord ↗

Step 2 — Complete workspace setup

Once the bot joins your server, a workspace is created automatically. To claim ownership you need your server's Guild ID.

To find your Guild ID: enable Developer Mode in Discord (User Settings → Advanced → Developer Mode), then right-click your server icon and choose Copy Server ID.
