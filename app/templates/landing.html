{# ── Section 8: FAQ ───────────────────────────────────────────────────── #}

Questions

Common questions

Is IronkeepV2 specific to Albion Online?

Yes. IronkeepV2 is built around Albion Online CTA coordination — fixed party sizes, role families (tank, healer, DPS, support), weapon-based builds, and structured signups. It is not a generic guild management tool.

How does the Discord integration work?

Officers post operation announcements and roster embeds to a Discord channel from within IronkeepV2. Players confirm attendance using a button on the Discord embed — the confirmation registers in the platform and updates the slot's readiness state. Discord handles communication; IronkeepV2 tracks coordination state.

Is it self-hosted?

Yes. You run IronkeepV2 on your own infrastructure. Your guild's operational data — compositions, slot assignments, attendance records — stays on your server. There is no hosted cloud version.

What kind of guild is this built for?

Guilds running structured CTA operations — ZvZ, HCE, faction warfare, hellgates — where composition planning and attendance are part of the officer workflow. If coordination currently lives in Discord threads and spreadsheets, IronkeepV2 is built for that situation.

Can multiple officers manage operations?

Yes. Officers share a workspace. Multiple officers can manage operations, update compositions, assign players, and post to Discord simultaneously.

Is there a mobile app?

Not currently. IronkeepV2 is a web-based tool built for desktop use. The interface is accessible from mobile browsers but is not optimized for mobile planning workflows.

Is IronkeepV2 actively maintained?

Yes. IronkeepV2 is in active development. The roadmap is prioritized around real coordination workflows used in actual guild operations.

How does player check-in work?

Officers post a roster embed to Discord with a check-in button attached. Players click the button in Discord — the check-in registers in IronkeepV2 and updates the slot's readiness state on the planner. No separate form, no manual tracking.
